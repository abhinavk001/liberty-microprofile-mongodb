How to setup.

1. Clone this repository.
2. cd into the folder.
3. Create a file named microprofile-config.properties at location src/main/webapp/META-INF/
4. Copy the following contents into the file and complete it with your mongoDB auth details.

mongo.dbname=yourdatabasename
mongo.user=dbusername
mongo.pass=dbpassword

5. Run the application.
